- [x] Ir pievienota klikšķināšanas skaņa
- [x] Var aiztaisīt projektu ar pogas nospiešanu
- [X] Var mainit cilvēka platumu un augstumu
- [X] Var izveleties cilvekus
- [X] Var apreķinat cik gadus vecs cilvēks ir ja ievada vardu un dzimšanas gadu
- [X] Pievienots drag script

- [X] Pievienotas vairākas drēbju kategorijas
